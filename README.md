Odoo Vim Editor
=====================

Installation
------------

Install packages:
- vim

```bash
sudo apt install vim-gtk # or your preffered

ln -s ~/path/to/this/vimrc ~/.vimrc
ln -s ~/path/to/this/vim ~/.vim
```
